# Typography

The default font family is Open Sans, with sans-serif as a fallback.

| Font property | Value      |
|--------------|------------|
| Family       | Open Sans  |
| Fallback     | sans-serif |
| Base size    | 16px       |
| Line height  | 1.4        |

---

## Typography System

The typography system is based on **semantic roles**, not raw sizes.

Helpers define:

- font size
- line height
- font weight

They **do not define color**.

---
